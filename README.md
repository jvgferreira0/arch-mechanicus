# arch-mechanicus

arch-mechanicus is a beautiful, modern & opinionated Arch Linux distribution.

Forked from Omarchy with customized themes, packages, and configurations.

## About

This distribution maintains the excellent foundation of Omarchy while:
- Using official Arch Linux mirrors (no third-party infrastructure)
- Custom theme selection and configurations
- Curated package selection
- Removing DHH-specific defaults
